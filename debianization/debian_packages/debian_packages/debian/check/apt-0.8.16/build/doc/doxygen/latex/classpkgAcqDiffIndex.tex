\subsection{\-Detailed \-Description}
\-An item that is responsible for fetching an index file of \{\{\{ package list diffs and starting the package list's download. 

\-This item downloads the \-Index file and parses it, then enqueues additional downloads of either the individual patches (using \doxyref{pkg\-Acq\-Index\-Diffs}{p.}{classpkgAcqIndexDiffs}) or the entire \-Packages file (using \doxyref{pkg\-Acq\-Index}{p.}{classpkgAcqIndex}).
