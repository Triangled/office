\subsection{\-Detailed \-Description}
\-An acquire item that is responsible for fetching a \{\{\{ translated index file. 

\-The only difference from \doxyref{pkg\-Acq\-Index}{p.}{classpkgAcqIndex} is that transient failures are suppressed\-: no error occurs if the translated index file is missing.
