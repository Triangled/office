\subsection{\-Detailed \-Description}
\-Retrieve an arbitrary file to the current directory. \{\{\{. 

\-The file is retrieved even if it is accessed via a \-U\-R\-L type that normally is a \-N\-O\-P, such as \char`\"{}file\char`\"{}. \-If the download fails, the partial file is renamed to get a \char`\"{}.\-F\-A\-I\-L\-E\-D\char`\"{} extension.
