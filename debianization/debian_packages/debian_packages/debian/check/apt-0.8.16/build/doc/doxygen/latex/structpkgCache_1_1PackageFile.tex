\subsection{\-Detailed \-Description}
stores information about the files used to generate the cache 

\doxyref{\-Package}{p.}{structpkgCache_1_1Package} files are referenced by \doxyref{\-Version}{p.}{structpkgCache_1_1Version} structures to be able to know after the generation still from which \-Packages file includes this \doxyref{\-Version}{p.}{structpkgCache_1_1Version} as we need this information later on e.\-g. for pinning.
